# fenssel-bucket

[![Tests](https://github.com/classic-mcfly/fenssel-bucket/actions/workflows/ci.yml/badge.svg)](https://github.com/classic-mcfly/fenssel-bucket/actions/workflows/ci.yml) [![Excavator](https://github.com/classic-mcfly/fenssel-bucket/actions/workflows/excavator.yml/badge.svg)](https://github.com/classic-mcfly/fenssel-bucket/actions/workflows/excavator.yml)

A custom [Scoop](https://scoop.sh) bucket with app manifests not available in the official buckets.

## How do I add this bucket?

```pwsh
scoop bucket add fenssel-bucket https://github.com/classic-mcfly/fenssel-bucket
```

## How do I install an app?

```pwsh
scoop install fenssel-bucket/<app-name>
```

## Apps

| App | Version | Description |
|-----|---------|-------------|
| deezer | 7.1.130 | Official Deezer music streaming desktop application |
